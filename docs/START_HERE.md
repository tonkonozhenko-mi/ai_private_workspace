# AI Private Workspace — start here

This is the outside-the-app entry point: a short map to the right doc for what
you want to do. Once the app is open, its own guided setup takes over.

## Pick your path

- **I just want to run it.** → [INSTALL_WALKTHROUGH.md](INSTALL_WALKTHROUGH.md) —
  download, open, and get to your first question. Everything runs locally; no
  cloud, no accounts.
- **I want to run it from source / develop on it.** →
  [DEVELOPMENT.md](DEVELOPMENT.md) — start the backend and frontend, run the
  tests, and understand the local workflow.
- **I want to contribute a change.** → [../CONTRIBUTING.md](../CONTRIBUTING.md) —
  how to propose changes, the review process, and the ground rules.
- **I want to understand how it's built.** → [ARCHITECTURE.md](ARCHITECTURE.md) —
  the ports-and-adapters backend, the desktop shell, and how the pieces fit.

For a feature overview and screenshots, see the [repository README](../README.md).
